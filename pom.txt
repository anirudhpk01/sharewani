Instead of:
JavaScript
db.collection.updateOne(
  { _id: id },
  { $set: { status: "closed" } }
)
Do:
JavaScript
db.collection.updateOne(
  { _id: id },
  { 
    $set: { 
      status: "closed",
      conversationId: existing.conversationId,
      market: existing.market,
      createdDateTime: existing.createdDateTime
    }
  }
)
